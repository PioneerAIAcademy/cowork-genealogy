- Tree read: given a person ID, return all information about that person found in the FamilySearch tree, including sources
- Research guidance: given a country name, return a markdown-formatted guide for how to do research in that country
- Online records: given a country name, return a list of online record sources for that country
- Wiki Query: given a question about how to do genealogy research, return relevant sections from the FamilySearch wiki as a list of markdown texts
- Wiki read: given a FamilySearch wiki page title, return the entire markdown for that page <- includes library resources and learning center videos
- Wikipedia Query: given a search, use the Wikipedia API to find relevant pages titles on Wikipedia
- Wikipedia Read: given a page title on wikipedia, return the text of that page
- Place Query: given a place name, return information about the place, including its id and its jurisdictional hierarchy over time
- Population Statistics: given a place id and time period, return population statistics about the number of people living in that place during that time period
- Collections Search: given a place id, return the collections at FamilySearch that cover that place
- External links: given a place id, return links to record collections or other information outside of FamilySearch about that place
- Record Search: given a structured genealogy search (names, dates, places, and relationships about a person and optional collection id), return matching records, including record ids, from FamilySearch historical records
- Record Read: given a record id, return all information about that record, including its image id
- Image search: given a structured genealogy metadata search (date, place, and required collection id), return the matching image ids
- Image transcription: given an image id, return the image transcription
- FullText Search: given a textual genealogy search query, return matching full-text chunks, including page ids, from FamilySearch full texts
- FullText Read: given a page id, return the full page text
- Match: given data (names, dates, places, and relationships) about one person or record and a second person or record, return the likelihood that they are the same person (based on an ML model)
- Warnings: given data (names, dates, places, and relationships) about a person, return warnings if the data is likely incorrect; e.g., the person married before 12 years old, died after 120 years old
- Distance calculator: given two place ids, return the distance between them
- Calendar converter - Julian↔Gregorian, Old Style/New Style, Quaker double-dating. Country-specific transition dates (England 1752, etc.)
- Schema validator — given a project file, validate against the published schema and return errors. Cheap to build, prevents an entire class of silent corruption.
