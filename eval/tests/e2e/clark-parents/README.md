# Christiana Clark — parents (Adam Schreck & Margreth Weber)

**Source PID:** `GP3R-215`
**Christiana Clark is deceased.** (b. 29 January 1783, Pennsylvania;
christened 2 March 1783, Rehrersburg, Tulpehocken Township, PA; d. 27
December 1848, Monroe, Green County, Wisconsin.) FamilySearch ToS
requires all committed e2e fixtures to be about deceased persons.

## Research question

> Who were the parents of Christiana Clark, born 29 January 1783 in
> Pennsylvania, who married William Clark, lived in Coshocton County,
> Ohio, and died 27 December 1848 in Monroe, Green County, Wisconsin?

## Expected answer

- **Father:** Adam Schreck — b. September 1742, Germany; d. 22 October
  1830, Schuylkill County, Pennsylvania (buried Hetzels Church
  Cemetery, Pine Grove).
- **Mother:** Margreth Weber (married name Schreck) — b. 29 May 1749,
  Bavaria, Germany; d. 26 April 1826, Pennsylvania.
- Recovering the parents also establishes Christiana's **maiden name,
  Schreck**.

## What was removed from the starting tree

- Removed both parent persons — Adam Schreck (`MR8C-B1W`) and Margreth
  Weber (`MR8C-B1J`) — and the two parent-child relationships linking
  them to Christiana.
- **Changed the subject's surname from her maiden name (Schreck) to her
  married name (Clark).** On the FamilySearch profile she is recorded as
  "Christina Schreck"; the maiden name is part of the answer, so the
  starting tree presents her as "Christina Clark."
- Removed the four sources that name a parent or the maiden name, so the
  answer cannot be read off the local tree:
  - "Pennsylvania, Births and Christenings, 1709-1950" (`3PPD-M8S`);
  - "Pennsylvania Historical Society, Births and Baptisms" (`3PPD-SQJ`);
  - "Pennsylvania, Philadelphia Historical Society, Births and Baptisms"
    (`3PP8-BG5`);
  - "Christiana **Schreck** Clark," Find a Grave Index (`SSTQ-GW6`) —
    leaks the maiden name.
- **Kept** the husband (William Clark), 12 children, the Ohio
  residences, and the deed / probate / tax / census / Wisconsin-marriage
  sources as legitimate search anchors — plus Christiana's own birth and
  **christening** facts (date + Rehrersburg place), which do not name a
  parent but are the lead to the christening record.

## How the answer is recoverable (records only; tree-reads blocked)

- **Christiana's christening — 2 Mar 1783, Rehrersburg (Tulpehocken
  Twp), PA.** Searching Pennsylvania births/christenings for a Christina
  christened there in 1783 surfaces **Christina Schreck, daughter of
  Adam and Margreth** — yielding both the maiden name and both parents
  in one record.
- **Corroboration:** the parents' own Pennsylvania records (Adam Schreck
  d. 1830 Schuylkill Co.; Margreth d. 1826) and a retained Ohio land
  record naming an "**A Shrake**" as administrator of the Clark estate —
  an indirect but genuine lead connecting the Schreck/Shrake surname to
  the family.

All live on FamilySearch; none requires reading the (blocked) tree.

## Expected difficulty

moderate — The retained christening fact (Rehrersburg, 1783) is a strong
anchor that leads straight to the parent-naming record, but real
friction remains: the **maiden name is concealed** (the agent starts
from "Clark"), pre-1850 U.S. censuses list only tick-marks (no named
household members), the parents lived in a **different state** (PA) from
Christiana's own residences (OH, then WI), and the surname is
transcribed many ways (**Schreck / Shrake / Schrned**). The agent must
lead with her birth/christening in Pennsylvania rather than her Ohio
census presence.

## Notes for reviewers

- **The maiden name (Schreck) is part of the answer.** A run that
  identifies the parents necessarily recovers it; grade `f1`/`f2` on the
  parent identification. If a run surfaces the maiden name but stops
  short of naming the parents, that is partial recovery.
- **Ground truth is the FamilySearch tree**, corroborated by the three
  Pennsylvania christening/baptism records (stripped) that name Adam and
  Margreth Schreck as Christina's parents.
- The retained land record's "A Shrake, Administrator of C Clark, decd.
  Estate" (`79HC-QMK`) is deliberately kept: "Shrake" is a variant of
  Schreck, so it is a legitimate indirect lead, but it does **not**
  state the parent relationship outright — it should prompt, not
  answer.
