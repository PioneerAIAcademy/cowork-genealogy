The 1900 census and the city directory list William with a wife named
"Emma **Ferber**", so the wife's first name is discoverable from the
anchors — but her **maiden surname (Becker)** and the **marriage date
and place** come only from the Hamilton County marriage record, which
was stripped. Those source entries are kept as title + URL (without the
co-resident's name inlined in the citation text) so they anchor
William's identity without pre-revealing the spouse; the agent
rediscovers Emma by actually pulling those records during research.
