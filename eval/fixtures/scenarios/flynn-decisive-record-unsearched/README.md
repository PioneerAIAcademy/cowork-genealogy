# flynn-decisive-record-unsearched

Delta of `flynn-census-exhausted`: q_001 (parentage of Patrick Flynn) has a
fully **completed, census-only** plan — the vital-record and probate plan
items were removed along with their searches and downstream entries. The
tree additionally cites Patrick's 1908 Schuylkill County death certificate
(source `S5`) which has never been examined in this project.

Built for the research-exhaustiveness rule *“named decisive records gate
the declaration”*: every plan item is complete, so a completion-biased
evaluation would declare — but the decisive record types for a parentage
question (subject's death record, baptism, parent's probate) are
unsearched, and one of them is literally cited in the tree.
