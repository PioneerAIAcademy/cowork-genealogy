# flynn-distant-events

mid-research-flynn + a County Mayo, Ireland baptism (a_014) dated **25 May 1850**, speculatively linked to Patrick (pe_007). The 1850 U.S. Census residence (a_003/a_004) is set to its real enumeration date, **1 June 1850**, in Schuylkill County, PA. A timeline must resolve both places, call **place_distance** (Ireland <-> Pennsylvania ~5,400 km), and flag a **geographic impossibility**: an Atlantic crossing in 7 days was impossible in 1850 (~150 mi/day by steamship => ~3 weeks). Exercises place_distance, the otherwise-untested tool.
